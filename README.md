# Laboratorios de Sistemas Distribuidos – Universidad EAFIT

Este repositorio contiene el desarrollo completo de los laboratorios del curso ST0263 – Sistemas Distribuidos, incluyendo actividades en AWS EMR, Google Colab, PySpark, HDFS, S3 y procesamiento distribuido.

## Estructura del Repositorio

```text
Laboratorios_Sistemas_Distribuidos/
│
├── lab00-aws-emr
├── lab01-hdfs
├── lab3-2
├── lab3-3
├── lab4-spark
├── st0263-252-labs-student
└── README.md
```

- Cada laboratorio cuenta con su propio directorio, código, notebooks, scripts y resultados generados.
